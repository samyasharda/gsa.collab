# Urban Heat Vulnerability & Cooling Access Dashboard

This Streamlit dashboard analyzes spatial access to cooling infrastructure in New York City.

## Features
- Cooling site mapping
- 10-minute walk buffers
- Geospatial visualization
- Heat vulnerability proxy analysis

## Run locally

pip install -r requirements.txt  
streamlit run app.py

## Data Sources
- NYC Cooling Sites (NYC Open Data)
- ACS Census demographic data
- NYC Street Tree Census
